# **App Name**: LlamaIA

## Core Features:

- Conversational AI: Provides a conversational interface powered by a custom AI model, allowing users to interact with the app via text or voice. The model can also access external tools, such as APIs.
- Stylized Llama Avatar: A central animated 3D llama avatar expresses different states (thinking, speaking, listening) based on the conversation.
- Chat History: Displays past conversations in a scrollable list using frosted glass cards. Also allows starting new conversations.
- Voice/Text Controls: Floating buttons offer controls to switch between voice and text input, as well as accessing settings and account information.
- Text Input Area: A futuristic text input area at the bottom for composing and sending messages.

## Style Guidelines:

- Primary color: Neon blue (#00F3FF) provides a futuristic, vibrant accent.
- Background color: Dark gradient from #0A0E17 to #121826 to set a futuristic and immersive scene.
- Accent color: Holographic purple (#9D4EDD) adds a sophisticated and high-tech touch.
- Font: 'Inter', a sans-serif for a modern and technological look in the UI.
- Subtle animations and particle effects enhance the sense of a dynamic, futuristic environment.
- Animated microphone that illuminates when activated.
- Frosted glass effect used to ensure focus remains on llama avatar.
